# 恆春墾丁民宿網站

## 本機預覽
直接開啟 `index.html`。

## GitHub + Vercel 部署
1. 將本資料夾內所有檔案上傳到 GitHub Repository 根目錄。
2. Vercel 選擇 Add New → Project。
3. 匯入 GitHub Repository。
4. Framework Preset 選 Other。
5. Build Command 與 Output Directory 保持空白。
6. 點擊 Deploy。

本網站為純 HTML、CSS、JavaScript，不需要 npm 或後端。
